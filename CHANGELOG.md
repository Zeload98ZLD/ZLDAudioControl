# Changelog

## v1.0.5 – Audio Source Engine

- Auswahl basiert jetzt direkt auf Windows-Audiositzungen
- Alle aktuell erkannten Audioquellen werden angezeigt
- Aktueller Pegel jeder Audioquelle
- Lautstärke und Mute-Status in der Auswahl
- Mehrere Sitzungen desselben Prozesses werden zusammengefasst
- Audioquellen können direkt aus dem Windows-Lautstärkemixer übernommen werden
- Aktualisieren-Button im Audioquellen-Dialog
- Spiele erscheinen, sobald sie eine Windows-Audiositzung erzeugen
- Auto-Game verwendet zuerst das aktive Fenster
- Auto-Game nutzt als Rückfallebene die lauteste geeignete Audioquelle
- Browser, Discord und Spotify bleiben als normale Audioquellen zuweisbar
- Bestehende MIDI-Zuweisungen, Profile, Themes, EQ und Updater bleiben erhalten
